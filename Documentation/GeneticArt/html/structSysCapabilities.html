Detailed Description

Holds flags for detected CPU/Platform capabilities.

This structure tracks the presence of various SIMD instructions (SSE/AVX), availability of OpenGL, OpenCL, and the maximum number of CPU threads.

The mutex field protects these flags in multi-threaded contexts.
